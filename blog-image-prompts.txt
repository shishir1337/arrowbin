BLOG IMAGE PROMPTS - manual generation list
=============================================

All 27 raster (.webp) blog images. The auto-generated versions were low
quality and have been deleted, along with the generator script
(scripts/generate-blog-images.mjs).

Every prompt below is written for Google Gemini 'Nano Banana' using its
official prompt structure, ordered most-important-first:
  Subject -> Action/relationship -> Setting -> Style/medium ->
  Composition/camera -> Lighting/color -> Key details -> Constraints.

Brand palette (kept consistent across every image):
  - Deep charcoal: #0a0a0a   - Lime-green highlight: #a3e635
  - Lime-green solid: #65a30d
Hero/cover images are 1600x900; all in-body infographics and photos are
1280x720 (both 16:9). Generate at that aspect ratio.

Generate each image, then save it at the exact "Save to" path below (keep
the same filename). The .svg files in these folders are data charts (kept,
NOT in this list).

Every entry is currently flagged `pending` in src/lib/blog.ts, so the site
shows a placeholder card instead of a broken image. When you hand the images
back, dropping each file at its path is enough - the pending flag is removed
so the real image renders.
